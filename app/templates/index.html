Problem Setup

The task is to predict a country's primary energy consumption (in TWh) given macroeconomic indicators, fuel production figures, and electricity mix data. Models are trained on 1990–2012 data and evaluated on the held-out 2018–2022 test set using a time-based split to avoid look-ahead bias.

Target variable is log(primary_energy_consumption + 1) during training — predictions are inverted to TWh for evaluation and display.
Data Pipeline

The raw dataset (22,012 rows, 129 columns) was cleaned down to a usable 12-feature matrix through:

  • Temporal filter → 1990–2022
  • Entity filter → real countries only (by ISO code)
  • Leakage removal → no direct sub-components of target
  • KNN imputation (k=5) for remaining missing values
  • Log transforms for skewed features
