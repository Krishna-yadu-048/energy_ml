Key findings from the cleaned dataset. All plots are generated by notebooks/02_eda.ipynb and saved as static images.

Dataset scope: 218 countries, 1990–2022, after removing regional aggregates and rows with a missing target variable. KNN imputation (k=5) applied to remaining missing values.
1. Target Variable Distribution

Raw consumption is extremely right-skewed — a handful of large economies dominate. The log-transformed version is approximately normal and is used as the training target.

Key Takeaways for Modelling
  • Log transform is necessary for the target — raw scale causes linear models to over-fit on large economies.
  • Population and GDP are the strongest predictors, but are collinear — VIF analysis flags them. Tree-based models handle this naturally; Ridge uses L2 regularisation.
  • Share features are bounded [0, 100] and some are near-complementary (fossil + renewables ≈ 100), introducing mild redundancy.
  • Temporal trend is weak once country-level features are included — year_norm has a low correlation with the target after conditioning on other variables.
  • Fuel production features (coal, oil, gas) add signal for major producers but are zero for most countries — this sparsity benefits tree models over linear ones.
